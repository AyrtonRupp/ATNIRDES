# ATNIRDES - Página Corporativa

Bienvenido al repositorio del proyecto de página corporativa para **Atnirdes**, una empresa tecnológica. Este proyecto fue diseñado con el objetivo de reflejar la esencia y profesionalismo de la compañía, utilizando herramientas modernas y un diseño adaptado a los estándares actuales.

## 📋 Especificaciones del Proyecto

- **Tecnología Principal**: Construido con [Astro](https://astro.build/), un framework moderno para la construcción de sitios web estáticos y dinámicos.
- **Paleta de Colores**: La paleta principal está basada en `rgba(130, 28, 111, 0.7)`, aportando un toque elegante y distintivo.
- **Características Clave**:
  - Carrusel dinámico y atractivo.
  - Fondos estáticos que enfatizan el diseño.
  - Totalmente responsive para adaptarse a cualquier dispositivo.
- **Auditoría y Diseño**: Todo el diseño y las decisiones estéticas han sido cuidadosamente auditados y realizados con el objetivo de optimizar la experiencia del usuario.

## 🌟 Características del Diseño

- **Responsive Design**: La página está optimizada para dispositivos móviles, tablets y escritorios.
- **Interactividad**: Elementos dinámicos como el carrusel enriquecen la experiencia visual.
- **Accesibilidad**: Diseño amigable para todos los usuarios.

## 🚀 Instalación

1. Clona este repositorio en tu máquina local:
   ```bash
   git clone https://github.com/AyrtonRupp/atnirdes.git
2. cd atnirdes
3. npm install
4. npm run dev
5. Accede al proyecto en tu navegador en http://localhost:3000.
